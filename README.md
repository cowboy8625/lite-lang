# lite-lang

### Quick Start

`cargo run --release -- repl`
or
`cargo run --release -- --help`

For grammer of languages see [grammer](./GRAMMER.md).

#### Shell Commands start with -> :
- :exit ---------> exit program.
- :help ---------> Output this message.
- :clear --------> Clear shell screen.



#### Language Syntax:
```
hello = "Hello"
space = " "
world = "World"
print hello + space + world
```
